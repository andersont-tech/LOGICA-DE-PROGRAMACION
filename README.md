# LOGICA-DE-PROGRAMACION
*Definir el alcance del sistema a desarrollar*
El sistema implica la creación de un juego interactivo del ahorcado utilizando el lenguaje de programación Python en el cual el programa permitirá al usuario elegir un nivel de dificultad (1. fácil, 2.medio o 3.difícil), donde cada categoría incluirá un conjunto de palabras y un número específico de intentos. 
El sistema presentará la palabra oculta a través de guiones, pedirá al jugador que introduzca una letra, comprobará si la letra está en la palabra y actualizará el progreso del juego manteniendo un registro de las letras ya utilizadas de igual forma mostrará los intentos que quedan y se dibujara el dibujo del ahorcado a medida que el jugador comete errores llegando a la parte final donde se da a conocer si el usuario gana o pierde la partida del juego.

*Identificar el problema que se desea resolver*
El proyecto tiene como objetivo abordar la necesidad de crear una aplicación que, a través de un juego interactivo, facilite el fortalecimiento y la práctica de los conocimientos esenciales de programación que se ha ido adquiriendo durante estos módulos de aprendizaje.
Para entender cómo funcionan las estructuras condicionales, los ciclos repetitivos, las listas, las funciones y la validación de datos en Python, con la programación de aplicaciones creando ejercicios prácticos poniendo en práctica todos los conocimientos y con el juego del ahorcado proporciona un contexto apropiado para emplear estos conceptos de manera divertida y dinámica con la programación de este juego el usuario interactuara con el programa, escogiendo un nivel de dificultad, adivine palabras y obtenga una respuesta instantánea sobre sus fallas y aciertos. Esto ayudará la adquisición de competencias para solucionar problemas a través del código y el entendimiento de la lógica de programación.
